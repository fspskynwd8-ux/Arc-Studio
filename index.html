Über uns

Ein Projekt, eine Person, ein Ziel.

Arc Studio ist kein Unternehmen — es ist ein Nebenprojekt, das während der eigenen Ausbildung als Mechatroniker entstanden ist. Die Idee war simpel: Es gab keine App, die wirklich für den Prüfungsalltag gemacht war. Also wurde eine gebaut. VOLTIQ ist das Ergebnis davon — kostenlos, ohne Tracking, ohne Bullshit. Wenn dir die App hilft, freut mich das. Feedback ist immer willkommen.

FAQ

Häufige Fragen.

Ist VOLTIQ wirklich kostenlos?
Ja, komplett. Kein Abo, keine versteckten Kosten, kein Freemium-Modell. Die App ist und bleibt kostenlos.
Wie installiere ich die App (PWA)?
Öffne VOLTIQ im Browser (Chrome oder Safari), tippe auf „Teilen" (iOS) oder das Menü-Symbol (Android) und wähle „Zum Homescreen hinzufügen". Fertig — die App verhält sich dann wie eine native App.
Funktioniert das auch auf älteren Smartphones?
Ja. VOLTIQ läuft auf jedem Gerät mit einem modernen Browser — auch auf älteren Android-Phones oder iPhone 8+. Keine Mindestversion nötig.
Werden meine Daten irgendwo gespeichert?
Nein. Alle Daten (Lernfortschritt, Berichtsheft, Quizergebnisse) bleiben ausschließlich auf deinem Gerät im lokalen Browser-Speicher. Es gibt keinen Server, keine Cloud, kein Konto.
Kann ich VOLTIQ ohne Internet nutzen?
Ja. Einmal geladen, funktioniert die App komplett offline — auch im Keller der Werkstatt oder ohne WLAN.
Ich habe einen Fehler gefunden / eine Idee.
Super, meld dich! Schreib eine E-Mail an arc.studio@gmx.de oder öffne ein Issue auf GitHub. Feedback wird direkt umgesetzt.
