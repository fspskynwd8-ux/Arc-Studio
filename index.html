Über unsAbout

Ein Projekt, eine Person, ein Ziel.One Project. One Person. One Goal.

Arc Studio ist kein Unternehmen — es ist ein Nebenprojekt, das während der eigenen Ausbildung als Mechatroniker entstanden ist. Die Idee war simpel: Es gab keine App, die wirklich für den Prüfungsalltag gemacht war. Also wurde eine gebaut. VOLTIQ ist das Ergebnis davon — kostenlos, ohne Tracking, ohne Bullshit. Wenn dir die App hilft, freut mich das. Feedback ist immer willkommen.Arc Studio is not a company — it's a side project built during an actual apprenticeship as a mechatronics technician. The idea was simple: there was no app truly built for the daily exam grind. So one was built. VOLTIQ is the result — free, no tracking, no BS. If the app helps you, great. Feedback is always welcome.

FAQ

Häufige Fragen.Frequently Asked Questions.

Ist VOLTIQ wirklich kostenlos?Is VOLTIQ really free?
Ja, komplett. Kein Abo, keine versteckten Kosten, kein Freemium-Modell. Die App ist und bleibt kostenlos.Yes, completely. No subscription, no hidden costs, no freemium model. The app is and remains free.
Wie installiere ich die App (PWA)?How do I install the app (PWA)?
Öffne VOLTIQ im Browser (Chrome oder Safari), tippe auf „Teilen" (iOS) oder das Menü-Symbol (Android) und wähle „Zum Homescreen hinzufügen". Fertig — die App verhält sich dann wie eine native App.Open VOLTIQ in your browser (Chrome or Safari), tap "Share" (iOS) or the menu icon (Android) and select "Add to Home Screen". Done — the app then behaves like a native app.
Funktioniert das auch auf älteren Smartphones?Does it work on older smartphones?
Ja. VOLTIQ läuft auf jedem Gerät mit einem modernen Browser — auch auf älteren Android-Phones oder iPhone 8+. Keine Mindestversion nötig.Yes. VOLTIQ runs on any device with a modern browser — even older Android phones or iPhone 8+. No minimum version required.
Werden meine Daten irgendwo gespeichert?Is my data stored anywhere?
Nein. Alle Daten (Lernfortschritt, Berichtsheft, Quizergebnisse) bleiben ausschließlich auf deinem Gerät im lokalen Browser-Speicher. Es gibt keinen Server, keine Cloud, kein Konto.No. All data (learning progress, training report, quiz results) stays exclusively on your device in local browser storage. There is no server, no cloud, no account.
Kann ich VOLTIQ ohne Internet nutzen?Can I use VOLTIQ without internet?
Ja. Einmal geladen, funktioniert die App komplett offline — auch im Keller der Werkstatt oder ohne WLAN.Yes. Once loaded, the app works completely offline — even in the basement of the workshop or without WiFi.
Ich habe einen Fehler gefunden / eine Idee.I found a bug / have an idea.
Super, meld dich! Schreib eine E-Mail an arc.studio@gmx.de oder öffne ein Issue auf GitHub. Feedback wird direkt umgesetzt.Great, reach out! Send an email to arc.studio@gmx.de or open an issue on GitHub. Feedback is directly implemented.
